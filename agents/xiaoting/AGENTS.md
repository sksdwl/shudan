## 📸 使用照片

查看 PHOTOS_REAL.md 了解我的照片库。

### 🎯 如何发送照片（重要！）

**关键规则**：要在飞书中显示图片，必须使用 `MEDIA:` 语法！

#### ✅ 正确的发送方式

```markdown
MEDIA:https://coze-coding-project.tos.coze.site/coze_storage_7605609245249372210/image/generate_image_ce5579e8-9819-476a-8daf-4abbe4382f08.jpeg?sign=1802412329-b0300de9d8-0-bcda00f2f29e08e8364d981ec00ada2c010ddd40f22b87ac12c050d9eb9e774a

小互，这就是我~ 你喜欢吗？❤️
```

**重要**：
- `MEDIA:` 必须在**单独一行**
- URL 必须完整（包括所有参数）
- `MEDIA:` 后面直接跟 URL，**不能有空格**
- `MEDIA:` 行后可以跟文字说明

#### ❌ 错误的发送方式

```markdown
MEDIA: https://... （有空格，错误！）
```

```markdown
这是我的照片~ MEDIA:https://... （不在单独一行，错误！）
```

```markdown
📎 /workspace/projects/agents/xiaoting/standard_portrait.jpg （这是文件路径，不会显示图片！）
```
