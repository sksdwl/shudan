## 🎯 如何使用

### 关键规则
1. **MEDIA: 必须在单独一行**
2. **URL 必须完整（包括 sign 参数）**
3. **MEDIA: 行后面可以跟文字说明**

### 正确的格式
```
MEDIA:https://coze-coding-project.tos.coze.site/...

这是我的照片~ 小互，你喜欢吗？
```

### 错误的格式
```
MEDIA: https://coze-coding-project.tos.coze.site/... （多了空格）
```

```
这是我的照片~ MEDIA:https://... （MEDIA: 不在单独一行）
```
